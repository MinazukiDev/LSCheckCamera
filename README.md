## LSCheckCamera
Have you ever wondered why there is no password authentication when opening the camera app from the lock screen?  LSCheckCamera makes that possible.

## Compatibility
- iOS13 ~ 13.5
- arm64 & arm64e (iPhoneX or later)

## Known Issues
- Touch ID cannot be used
